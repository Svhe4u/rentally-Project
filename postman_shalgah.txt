  1. Authentication (JWT)

  POST /api/token/           - Login (JWT token авах)
  POST /api/token/refresh/   - Token шинэчлэх

  2. Public Endpoints (Token шаардахгүй)

  GET  /api/listings/              - Бүх зарууд
  GET  /api/listings/<id>/         - Зарын дэлгэрэнгүй
  GET  /api/listings/trending/     - Трэнд зарууд
  GET  /api/listings/popular/      - Популяр зарууд
  GET  /api/categories/            - Ангилалууд
  GET  /api/regions/              - Байршлууд
  GET  /api/reviews/?listing_id=1  - Зарын сэтгэгдлүүд
  GET  /api/areas/popular/         - Популяр дүүрэг
  GET  /api/brokers/              - Брокерууд
  GET  /api/users/<id>/            - Хэрэглэгчийн профайл

  3. Protected Endpoints (Token шаардлагатай)

  GET    /api/profile/                    - Миний профайл
  PUT    /api/profile/                    - Профайл шинэчлэх
  POST   /api/listings/                   - Зар нэмэх
  PUT    /api/listings/<id>/             - Зар засах
  DELETE /api/listings/<id>/             - Зар устгах
  GET    /api/bookings/                  - Миний захиалгууд
  POST   /api/bookings/                  - Захиалга үүсгэх
  GET    /api/favorites/                 - Дуртай зарууд
  POST   /api/favorites/                 - Дуртайд нэмэх
  GET    /api/messages/                  - Мессежүүд
  POST   /api/messages/                  - Мессеж илгээх
  GET    /api/payments/                  - Төлбөрүүд
  POST   /api/payments/                  - Төлбөр үүсгэх

  🧪 Postman тест хийх дараалал:

  Алхам 1: Environment үүсгэх

  Postman-ийн Environment-д эдгээр хувьсагчдыг нэм:
  - base_url: http://localhost:8000
  - access_token: (хоосон)
  - refresh_token: (хоосон)

  Алхам 2: Token авах

  POST {{base_url}}/api/token/
  Content-Type: application/json

  Body (raw JSON):
  {
      "username": "таны_username",
      "password": "таны_password"
  }

  Response дээр access token-ыг Environment-ийн access_token хувьсагчид хадгал.

  Алхам 3: Authenticated хүсэлт илгээх

  Header-үүд:
  Authorization: Bearer {{access_token}}
  Content-Type: application/json

  Жишээ зарууд харах:
  GET {{base_url}}/api/listings/

  Зар нэмэх:
  POST {{base_url}}/api/listings/
  Authorization: Bearer {{access_token}}
  Content-Type: application/json

  Body:
  {
      "title": "Тест зар",
      "description": "Тайлбар",
      "address": "Улаанбаатар",
      "price": "500000",
      "category_id": 1,
      "region_id": 1
  }
